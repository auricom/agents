## Installation

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Setup

```bash
# Clone the repository
git clone https://github.com/example/my-basic-project
cd my-basic-project

# Install dependencies
npm install
```

## Usage

```bash
npm start
```
